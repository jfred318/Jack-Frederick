Email Updater

The Idea

This is an email update on popular tech stocks that is sent via a Python file I connected to a Ec2 instance using Amazon Web Services. It delivers an email at 8am and 4pm everyday to me and acn be accessed in my github with instructions on how to use yourself.

What I'm working on

My next step with this project evolves around an implementation of AI. I think it would be very helpful for me to recieve recent articles and major news about each of these stocks. I vision AI helping me accomplish this goal in the near future.
